    • What is QuickTime and how does Vital use it?

    QuickTime is a multimedia framework developed by Apple Inc. capable of handling various formats of digital video, media clips, sound, text, animation, music, and several types of interactive panoramic images. Available for Classic Mac OS, Mac OS X and Microsoft Windows operating systems, it is the software that allows us to actually play the videos in Vital on your computer.

    QuickTime 7 or later is required to view the videos in VITAL. Please check your computer for this software and its version, and if necessary download a free version of QuickTime from Apple's QuickTime Web site. Please refer to this QuickTime help page to test the player and learn more about it.

    The videos may vary in quality depending on your network connection and how dated the source video is. In general, the more bandwidth you have, the smoother the videos will play, and the more recent the footage and format, the clearer the image will be.

    Getting Started With Quicktime for Windows and Macs

    QuickTime is available in two forms: QuickTime Player and QuickTime Pro. QuickTime Player is a free, downloadable application that allows you to view QuickTime multimedia on your computer. QuickTime Pro is a paid upgrade which unlocks media creation and editing functions to QuickTime Player. It is only necessary for you to have QuickTime Player to view multimedia on your computer.

    You will need to have the most recent version of QuickTime Player in order to hear and view the media files on this site. If you have never used QuickTime or need to install it, here are some tips to help you get started.

    NOTE: You will only need to the free QuickTime Player, not QuickTime Pro, in order to access multimedia files.

    Windows

    In most cases, QuickTime Player does not come preinstalled on Windows computers. To download the most recent version of QuickTime, please visit the Quicktime Homepage. Your PC must meet the following requirements to run Quicktime:

    1. Pentium processor-based PC or compatible computer
    2. Windows 2000/XP
    3. At least 128MB of RAM

    In order for Web video to operate properly on your PC, the following settings must be verified:

    1. From the Start menu, go to 'Contol Panel'.
    2. Select 'QuickTime'.
    3. From the 'Browser' tab, select 'Play movies automatically'.
    4. From the 'Streaming' tab, select 'Automatic' form the Streaming Speed drop down menu and 'Enable Instant-On'.
    5. From the 'Advanced' tab, select 'Automatic' form the Transport Setup drop down menu.

    Updating QuickTime Player

    If you already have QuickTime Player on your computer but are not sure if it is the latest version, there is an easy way to both check and upgrade at the same time without downloading an entirely new version:

    1. Be sure that you are connected to the Internet.
    2. From the Start menu, go to 'Control Panel'.
    3. Select 'QuickTime'.
    4. From the 'Update' tab, select 'Update or install Apple QuickTime software'.
    5. To make sure you are notified of future QuickTime upgrades, you can select the box for 'Check for updates automatically'

    Macs

    Your Mac must meet the following requirements:

    1. QuickTime version 7 or higher
    2. Mac OS X v. 10.3 or later
    3. PowerPC (400 MHz or higher) or Intel based processor
    4. At least 128 MB of RAM

    QuickTime is built into the architecture of the Mac OS X operating system. If you are running any version of Mac OS X (10.1 or above), you will not need to install QuickTime, but you may want to make sure the software is up to date.

    In order for Web video to operate properly, the following settings must be verified:

    1. Go to 'System Preferences'.
    2. Under 'Internet & Network', select 'QuickTime'.
    3. From the 'Browser' tab, select 'Play movies automatically'.
    4. From the 'Streaming' tab, select 'Automatic' form the Streaming Speed drop down menu and "Enable Instant-On".
    5. From the 'Advanced' tab, select 'Automatic' form the Transport Setup drop down menu.

    2. Go to 'System Preferences'.
    3. Under 'Internet & Network', select 'QuickTime'.
    4. From the 'Update' tab, select 'Update or install Apple QuickTime software'.
    5. To make sure you are notified of future QuickTime upgrades, you can select the box for 'Check for updates automatically'.

    Troubleshooting QuickTime on Windows and "Switching Transports"

    On Windows computers, QuickTime streaming can be affected by the firewall settings on your personal computer. This problem can be diagnosed if your QuickTime plug-in within VITAL presents a "Switching Transports" message, followed by no immediate playback of video materials.

    If you suspect that your firewall is preventing access to streaming video clips, try configuring your computer in either of these two alternative ways:

    Method 1: Windows Firewall

    QuickTime streams are typically transported via the RTP/RTSP protocol, but many firewalls (including the Windows XP Firewall) often block these protocols. These ports settings can be changed in 'Control Panel > Windows Firewall' panel:

    * Click on the 'Exceptions' tab, and continue to select 'Add Port'. Proceed to add the ports and port information specific to QuickTime:

    1. Open port 80 for RTSP/TCP data
    2. Open port 554 for RTSP/TCP data
    3. Open ports 6970 through 6999 (inclusive) for RTP/UDP data.

    Alternatively, you may also choose to make your default web browser an excepted program, as well. simply click the selection box for your web browser.

    Method 2: QuickTime Settings

    1. From the Start menu, select Control Panel.
    2. In the Control Panel, find and double click the icon for the 'QuickTime'.
    3. Select the 'Advanced' tab.
    4. Find the 'Streaming' panel, and the drop down menu next to 'Transport Setup'.
    5. Change drop down selection from 'Automatic' to 'Custom'. Here, tell QuickTime to use 'HTTP', port '80'.
    6. Select 'OK'
    7. If the problem persists, repeat the above steps, but alternately use 'UDP', port 554.

    If the problem is not resolved by any of the above steps, please contact ccnmtl-vital@columbia.edu
